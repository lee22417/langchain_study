## langchain_study

## Env

- langchain, openai, Streamlit, python3, venv, VS code
- MacBook Air (Apple M2, Sonoma 14.0)

## Installation

```bash
// 패키지 설치
pip install -r requirements.txt

// 가상화 생성, 활성화
python3 -m venv env
source env/bin/activate

// streamlit local 실행
streamlit run home.py (http://localhost:8501)
```
